# Literature Review and Competitor Analysis Notes

## Literature themes

1. Classical pairs trading and statistical arbitrage.
2. Cointegration and error-correction models.
3. Machine learning for trade filtering and tabular decision models.
4. Deep learning for sequential spread/regime behavior.
5. Synthetic-data backtesting and robustness evaluation.

## Competitor / benchmark categories

| Category | Example | Relevance to project |
|---|---|---|
| Classical distance pairs trading | Gatev, Goetzmann, and Rouwenhorst-style distance approach | Baseline benchmark. |
| Cointegration / ECM strategy | Engle-Granger residual and ECM framework | Stronger econometric baseline. |
| Backtesting frameworks | Backtrader, QuantConnect LEAN | Infrastructure comparison. |
| ML-enhanced stat-arb | Logistic regression, random forest, XGBoost filters | Proposed value-add. |
| Synthetic robustness testing | López de Prado-style synthetic scenarios | Robustness contribution. |
